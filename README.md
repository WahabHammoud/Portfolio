# Wahab Hammoud - AI Engineer Portfolio

A modern, animated portfolio showcasing AI & Data Engineering expertise.

## Tech Stack

- Next.js
- Three.js
- Framer Motion
- Tailwind CSS

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
